Publishing path

Use this build as the polished front layer first, then connect the advanced systems in stages so the brand can go live without waiting for a full custom platform stack.

1. Publish

Upload `index.html`, `manifest.webmanifest`, and your brand assets to a GitHub Pages repository.

2. Brand

Attach a custom domain once you secure the name you want to own.

3. Connect

Add AI through Perplexity API and account-managed controls outside the static layer.

4. Expand

Move into real auth, gated rooms, payments, and protected creator tools with a backend.
